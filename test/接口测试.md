## 配置单个sshkey

* 生成key `ssh-keygen -t rsa -C <email>`
* cd ~/.ssh
* 复制 id_rsa.pub 公钥内容到 github

## 配置多个sshkey

```
cd ~/.ssh
vim config

Host github.com
HostName github.com
User nickyzj
IdentityFile /Users/zhengji/.ssh/id_rsa
```
